Aprender a ser um desenvolvedor

Aprender a ser um desenvolvedor é muito mais do que aprender uma linguagem nova. Envolve o aprendizado sobre as ferramentas e como realizar um setup de sucesso, isto é, configurar adequadamente cada uma, como por exemplo Git, VS Code e plugins, Slack, Zoom, dentre outras. Além disso, os conceitos de Unix e Internet foram assimilados, bem como o uso do Git via Bash.
